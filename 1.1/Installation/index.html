    Installation

    Windows 10 or higher

    The software can be downloaded from MS Sharepoint after logging in with your UW account. If you are a member of the Vascular Imaging Lab, but do not have download permission, please contact Xin Wang.

    After download you will get a zip file. Unzip it (password is mocha) and then you can open the software by double-clicking an executable file named mocha.exe in the root folder.

    MacOS or Linux

    Please contact Xin Wang for a workable version of the software on these operating systems. A download link will be available on this webpage in the future.
